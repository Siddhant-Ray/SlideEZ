1. Installing python

* For Windows, download python 3.6 from the official site
* For Linux or Mac, use 

$ sudo add-apt-repository ppa:jonathonf/python-3.6

$ sudo apt-get update
$ sudo apt-get install python3.6

$ sudo apt-get install python 3.6

$ sudo update-alternatives --install /usr/bin/python3 python3 /usr/bin/python3.5 1

$ sudo update-alternatives --install /usr/bin/python3 python3 /usr/bin/python3.6 2

$ sudo update-alternatives --config python3

$ export PATH="$PATH:/usr/local/bin/python

2. Installing dependencies-

a) EEL framework

* pip install eel 

b) TextRazor

* pip install textrazor
 
c) DeepSegment

* pip install --upgrade deepsegment

d) PPTx 

* pip install python-pptx==0.2.6


3. Run the application

* Run python mainApp.py from the CLI

* Click on New Session

* Click on Add Point and speak into the microphone

* Voila! Study the logs and when the execution is over, open testppt3.pptx
